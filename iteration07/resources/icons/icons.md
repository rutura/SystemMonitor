# System Monitor Icons

## Required Icons

### Application Icon
- **Filename**: `app-icon.svg`
- **Description**: Main application icon - system monitoring dashboard/gauge
- **Sizes needed**: 16x16, 32x32, 48x48, 256x256 (for .ico conversion)

### CPU Icon
- **Filename**: `cpu.svg`
- **Description**: Microchip/processor icon

### Memory Icon
- **Filename**: `memory.svg`
- **Description**: RAM stick or memory chip

### Disk Icon
- **Filename**: `disk.svg`
- **Description**: Hard drive or SSD (modern storage)

### Network Icon
- **Filename**: `network.svg`
- **Description**: Network connection (ethernet/wifi/network nodes)

## Specifications

- **Format**: SVG (preferred) or PNG
- **Style**: Monochromatic or dual-tone, clean line art
- **Theme**: Must work with both light and dark themes
